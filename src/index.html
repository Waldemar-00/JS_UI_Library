Method chaining, or simply chaining, in JavaScript can be defined as when one or more sequential methods get invoked from an object without the introduction of unnecessary variables. The sole purpose of chaining is to make our code more readable and reduce the redundancy within. If you are familiar with and methods from JavaScript promises then you already have some basic knowledge as to how chaining works.
Looking at the example above, we can see that we attempt to get the time zone offset from a date object. Look at how many variables we have declared here to return such a simple string. There are 8 to be exact. This seems very redundant for something as simple as this. So how can we make this better? With the help of chaining of course!
